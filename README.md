## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

#### Backend Setup
```bash
cd backend
npm install
```

#### Frontend Setup
```bash
cd frontend
npm install
```

### Running the Application

#### Start Backend
```bash
cd backend
npm start
# or for development
npm run dev
```

#### Start Frontend
```bash
cd frontend
npm start
# or for development
npm run dev
```
